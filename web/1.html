HTML

<Spurs vs Fullham>

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Since HTML defines the markup for a particular web page, you'll want the text, images, or other embeds to appear in certain ways.
For example, you might want some text to be big, other text to be small, and some to be bold, italic, or in bullet point form.
